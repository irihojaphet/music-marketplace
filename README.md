# Music Marketplace

A full-stack music marketplace built with FastAPI, PostgreSQL, React, React Query, Tailwind CSS, and Docker.

## Status
Project initialized. Dockerized PostgreSQL is running. Backend and frontend scaffolding are in progress.

## Stack
- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Pydantic
- JWT
- React
- React Query
- Tailwind CSS
- Docker Compose
